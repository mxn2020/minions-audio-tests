# Welcome to Minions Audio-tests

We're excited to introduce **Minions Audio-tests** — Voice interaction configs, TTS/STT pipeline settings, and audio samples.

## What's Included

- **TypeScript SDK**: `npm install @minions-audio-tests/sdk`
- **Python SDK**: `pip install minions-audio-tests`
- **CLI**: `npm install -g @minions-audio-tests/cli`

## Getting Started

Check out the [documentation](https://audio-tests.minions.help) for a full guide.
