# Minions Audio-tests

Voice interaction configs, TTS/STT pipeline settings, and audio samples

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-audio-tests/sdk`) — Core library for Node.js
- **Python SDK** (`minions-audio-tests`) — Python bindings
- **CLI Tool** (`@minions-audio-tests/cli`) — Command-line interface
- **Web App** — Interactive playground at [audio-tests.minions.wtf](https://audio-tests.minions.wtf)
