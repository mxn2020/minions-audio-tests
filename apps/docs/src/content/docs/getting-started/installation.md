---
title: Installation
description: Install Minions Audio-tests packages
---

## TypeScript / Node.js

```bash
npm install @minions-audio-tests/sdk minions-sdk
```

## Python

```bash
pip install minions-audio-tests
```

## CLI

```bash
npm install -g @minions-audio-tests/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
