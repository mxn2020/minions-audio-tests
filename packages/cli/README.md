# @minions-audio-tests/cli

CLI for Minions Audio-tests — Voice interaction configs, TTS/STT pipeline settings, and audio samples

## Installation

```bash
npm install -g @minions-audio-tests/cli
```

## Usage

```bash
audio-tests --help
```
